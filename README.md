# Digital Electronics Projects

A collection of digital electronics projects designed and simulated using **Logisim Evolution**. This repository contains basic combinational and sequential logic circuits for learning and demonstration purposes.

## Projects

### 1. 7-Segment Display Using NOT Gate
A simple circuit demonstrating how a NOT gate controls two 7-segment displays.

**Features**
- 1 Input Pin
- 1 NOT Gate
- 2 Seven-Segment Displays
- Demonstrates logical inversion
- Built in Logisim Evolution

**Project Folder**
```
7_segment_display/
```

---

### 2. 4-Bit Buffer Register
A 4-bit buffer register implemented using D Flip-Flops. Each flip-flop stores one bit of data and updates its output on the clock signal.

**Features**
- 4 D Flip-Flops
- 4 Data Inputs
- Shared Clock Signal
- 4 Output Bits
- Parallel Data Storage
- Built in Logisim Evolution

**Project Folder**
```
buffer_register/
```
